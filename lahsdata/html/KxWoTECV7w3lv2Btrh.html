套牢男人四大心法！你想知道的都在这里！

吸引男人容易，吃定男人难。


想要吃定一个男人，你真得有两把刷子。


我认识很多撩汉高手，情商高手，总结了她们的各种套路，总结了这四句心法：


  1. 物质上锁定

  2. 心理上依赖

  3. 事业上辅助

  4. 身体上和谐


什么意思呢？


  1. 让男人不断的对你进行物质和时间上的投入来增加他的沉没成本。

  2. 表面上你是依赖他的，其实他从心理上已经开始依赖你，离不开你。

  3. 事业方面你很厉害，但是比这个更厉害的是，你能辅助他，让他更厉害。（这是旺夫女必备技能，优秀男人都爱这种女人）

  4. 你们从身体上能非常的和谐，这是你们情感的坚实基础。


我的课程主要帮你提升1，2两条。


第三条，很多的女孩已经具备了。

第四条，靠你自己私下努力咯。


我们说说第一条，为什么一定要增加一个男人的沉没成本。


首先分析一下男人追女人的心态：


  1. 纯约炮

  2. 谈恋爱

  3. 想结婚


一.纯约炮：


男人一旦是想和你纯约炮的话，他一定是不想花钱的。

他的操作套路：


1.高价值展示：朋友圈装逼

2.语言上撩拨：目的就是约炮，各种给你洗脑。

3.尽量不花钱：约你去他家，或者去你家，省钱。

4.只约你上床：除了约你上床，不会主动联系你。


这种男人，你想套牢他？吃定他？门都没有！你中了这种套路，我也救不了，自认倒霉吧。



二.谈恋爱


男人如果想和你好好谈恋爱的话，他一定是话很多的。

他的操作套路：


  1. 不断的邀约：约你出去各种玩，不一定开房。

  2. 经常和你聊：和你在一起话特别多，聊不完。

  3. 总想对你好：买礼物，照顾你，带你见朋友。

  4. 对你说情话：约炮男不会对你说这种情话的！


这种男人是好好和你谈恋爱的，如果真的对你好，请你不要作，请你好好珍惜。


三.想结婚


男人如果想和你结婚过日子的话，他一定上来就是查户口的。

他的操作套路：


  1. 先查你户口：这是很多女孩超级反感的，但这是一个好信号，他很认真。

  2. 总是提结婚：他可能是家里催的急，特别想找个合适的人结婚。

  3. 想和你同居：同居是婚前最好的实验，同居不错，可以考虑结婚。

  4. 带你见父母：带你见父母就是要准备和你结婚的一个信号！


这种男人也许很无趣，但是他会一般来说会对你负责，一般出现于相亲男中。



从以上分析可以看出，越是认真的男人，越是会从物质上考虑对你的付出，更多的是考虑如何对你负责，和你结婚。然而不想和你恋爱和结婚的男人，只关注如何套路你和他上床。


所以想让一个男人长期和你在一起，物质上的锁定非常重要，让他对你不断进行物质和时间的付出，如果他愿意，说明他想发展长期关系；如果他不拒绝，说明他就只是玩一玩。


先我们说说第二条，表现上你要依赖男人，但是从心理上，你要让男人依赖你。


表面上依赖男人，实际上就是给男人面子，让男人发挥表现自己的一种方式。

表面上的无能与无用，让男人有帮助你的机会，展示自己能力的机会，男人会觉得和你在一起很有面子，自己像个英雄。


我经常说，无用的女人是最好的女人，是什么意思？


无用不是没有用，而是会而不用。


这种女人最旺夫。


这种女人什么都会，但是什么都不自己做，她们把自己的智慧给男人，让男人帮她们实现自己想要的一切，其实本来她们可以自己得到一切，她们却把这种机会给了自己的男人，实际上是一种双赢。这种女人就像是军师一样藏在男人身后，默默辅佐男人成功，这种女人最难得，男人对这种女人最依赖。


男人一旦依赖一个女人不断的给予他智慧和能量，将会形成强大的心理依赖。


那么其实第三条我也顺便解释了一下。


最后一条关于身体和谐方面，其实并不难，只要身体好，技巧到位，都是可以和谐的。最难的是第二条和第三条。
